# @haydywoo/dave-tokens

DAVE design tokens — colours, shadows, radii as CSS custom properties.

Source of truth for the DAVE design system. Used by [`@haydywoo/dave-react`](https://www.npmjs.com/package/@haydywoo/dave-react) and [`@haydywoo/dave-charts`](https://www.npmjs.com/package/@haydywoo/dave-charts).

## Install

```bash
pnpm add @haydywoo/dave-tokens
```

## Use

Import the CSS file into your global stylesheet:

```css
@import '@haydywoo/dave-tokens/dist/tokens.css';
```

This sets up all `--color-*`, `--shadow-*` custom properties that DAVE components depend on.
